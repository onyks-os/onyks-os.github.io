App Tesoreria

Il gestionale open-source per associazioni, collettivi e realtà no-profit. Sostituisce i complessi fogli Excel con un'interfaccia moderna e intuitiva, progettata per gestire conti comuni, debiti dei soci e riconciliazione bancaria senza richiedere conoscenze contabili avanzate.

Architettura & Database

Il sistema si basa su un'architettura Electron MVC-like per garantire sicurezza e persistenza offline. L'interfaccia React (Renderer) comunica con il database SQLite gestito in Node.js (Main) esclusivamente attraverso un bridge IPC sicuro.

ℹ DATA INTEGRITY
Better-SQLite3 sfrutta i prepared statements per prevenire injection. L'integrità referenziale è mantenuta tramite Foreign Keys e soft-delete dei membri per non corrompere lo storico contabile passato.
schema.sql
# Schema DB Relazionale (better-sqlite3)
📦 membri // Anagrafica soci
├─ id (PK, Autoinc)
├─ nome, cognome (Text)
└─ deleted_at (Soft delete)
📦 acquisti // Eventi finanziari
├─ id (PK)
├─ tipo (acquisto | raccolta_fondo | movimento)
├─ acconto_fornitore (Spese anticipate dal fondo)
└─ completato (0: Aperto, 1: Chiuso/Congelato)
📦 quote_membri // Tabella di giunzione (M:N)
├─ acquisto_id (FK -> acquisti)
├─ membro_id (FK -> membri)
├─ quantita (Moltiplicatore quota)
└─ importo_versato (Pagamento effettivo del membro)
📦 fondo_cassa // Registro movimentazioni extra
├─ importo (Positivo o Negativo)
└─ descrizione (Causale dell'operazione)

Quick Start

bash
# 1. Clona il repository ed entra nella directory
$ git clone https://github.com/onyks-os/App-Tesoreria.git
$ cd App-Tesoreria/app
# 2. Installa le dipendenze (Node.js 18+ richiesto)
$ npm install
# 3. Avvia in modalità sviluppo o compila l'eseguibile
$ npm run dev
$ npm run build
