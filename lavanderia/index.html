Lavanderia Galvani

A centralized, mobile-first booking system for washing machines and dryers at the Galvani student hall. Built to resolve scheduling conflicts between tenants, prevent usage monopolies, and ensure fair access to shared laundry resources — with a privacy-first, GDPR-compliant architecture.

Key Features

BOOKING LOGIC
  • → Pessimistic locking prevents double-booking under concurrent requests
  • → Dynamic weekly hour limits per user to prevent monopolies
  • → Server-side DateTime validation — no past slots, no future weeks
  • → Real-time delay reporting so the next tenant is warned
SECURITY & PRIVACY
  • → Registration restricted to @studio.unibo.it / @unibo.it domains
  • → Public identifiers are randomized usernames only — never real name or email
  • → PASSWORD_BCRYPT hashing + CSRF tokens on every form
  • → GDPR right-to-erasure: irreversible anonymization on request
UI / UX
  • → Mobile-first with thumb-zone navigation optimization
  • → Native dark mode — designed for low-light environments
  • → Toast notifications for instant feedback
  • → Full i18n — Italian 🇮🇹 and English 🇬🇧 out of the box
ADMIN PANEL
  • → Toggle machine status (active / maintenance) instantly
  • → Reset passwords and anonymize problematic accounts
  • → Adjust booking limits and post announcements without redeploying
  • → Rate-limit config hot-swap from the hidden Dev Console
0x02

Context

Shared laundry rooms in student halls have a classic coordination problem: no system, no fairness, constant conflict. Lavanderia-Galvani solves this with a lightweight web app that any tenant can access from their phone, requires no installation, and runs on standard shared hosting.

ℹ DESIGN PHILOSOPHY
Built to be dependency-free and deployable anywhere — no Composer, no NPM, no Docker. The entire stack runs on vanilla PHP 8 with a single index.php router. Adding a new language is one dictionary file. Moving to a different host is a file copy and a SQL import.
quick start
$ git clone https://github.com/onyks-os/Lavanderia-Galvani.git
$ mysql -u root -p lavanderia < database.sql
$ php -S localhost:8000
# open http://localhost:8000 — register with your @unibo.it email
